Quantum Programming Projects

Quantum Compilers and Circuit Optimization

While working as an intern at Rigetti Computing, I was tasked with working on the compiler and optimizer which processes Quil code, which then gets executed on Rigetti's quantum computers. This was a fun challenge for me, which involved thinking about and implementing control flow graph optimization, dead code elimination, quantum gate fusion, and algorithmic techniques for making the quantum virtual machine faster. The project also taught me about the usage of Common Lisp.

Writeup and more info coming soon!

FPGA Programming for NV Centers

Using a high-level FPGA programming interface, this project is geared toward the creation of a tool set for generating conditional pulse sequences. These pulse sequences will be used for nitrogen-vacancy center initialization and Rabi oscillations. See the papers section for more details.

See the code at github.com/vontell/artiq-control

Neural Network for NV-Center Classification

Using a neural network made with TensorFlow, and a small set of training data consisting of spectrum scans of possible nitrogen-vacancy centers in diamond, we were able to decide with decent accuracy whether or not the given scan was characteristic of a Nitrogen-vacancy center.

See the code at github.com/vontell/NV-NeuralNet

Quantum Algorithms

While working at Rigetti Quantum Computing as a Junior Quantum Engineer Intern, I helped to develop quantum algorithms using their open source platform Forest. I helped to implement algorithms such as Shor's algorithm, amplitude amplification, the quantum Fourier transform, superdense coding, and more.

See my contributions at github.com/rigetticomputing/grove

Race Against a Quantum Computer

Using a quantum virtual machine and the Quantum Approximate Optimization Algorithm, my friend Harley and I built a website to demo the power of Rigetti's new open source quantum programming platform, called Forest.

See the result at demo.rigetti.com

bloch-js

In part to teach others about quantum computers, my friend Joe and I built bloch-js, a simple JavaScript plugin for an interactive Bloch sphere. Using three.js, the applet features pannable interactions, arbitrary rotations, precession tracing, and data set simulation.

See the code at github.com/vontell/bloch-js

QStudio

In an attempt to make quantum programming easier for beginners, I began developing a quantum circuit IDE and learning resource, called QStudio. I did not get the time to finish this project yet, but it current allows for selecting and dropping Hadamard gates onto a circuit, adding extra qubits to this circuit, and also features a text editor for eventually editing .qasm and .quil code.

See the code at github.com/vontell/qstudio

Web Development Projects

Battlecode

Battlecode is MIT's longest running programming competition, with a new game developed every year. With over 3000 participants (which includes high school students, undergraduates, graduates, and international students), the competition requires the use of AI and strategy to beat another team at a certain task. My role as a developer on the team (after competing for a year as a participant) has been developing a new website and helping to plan the competition.

See more at battlecode.org

ErrorCode

ErrorCode is a code testing platform built during HackMIT 2016. The platform allows users to upload snippets of code which other users can write tests for, in a gamified way. Test cases are upvoted, and users win points based on their test reputation.

See the code at github.com/vontell/ErrorCode

Judgd

Using TensorFlow, my friend Cooper and I built a website powered by TensorFlow to scrape data from Devpost, a popular hackathon hosting website. After parsing this data with a beefy AWS instance, we were able to predict and view the probability of winning a hackathon based on the number of team members, technologies used, etc...

See the code at github.com/vontell/Judgd

Android Development Projects

IndicatorBinder

IndicatorBinder is an Android library which makes attaching indicators to your ViewPagers super simple. I started this project out of my own frustration for easily adding this feature to one of my applications. Available for download via Gradle.

See the code at github.com/vontell/IndicatorBinder

Accessibility for Mobile Applications

Under the Decentralized Information Group at MIT CSAIL, this project is devoted to analyzing the landscape of accessibility of native mobile applications. The project also involves working on a tool set and list of resources for developing accessible applications.

See the project at ama-csail.github.io

MathIt

MathIt is an Android application that allows friends to send eachother LaTeX through Facebook Messenger and other social media platforms. I started this project because my friends and I would always have trouble sending long equations through Facebook Messenger. The project uses MathJax to render the LaTeX.

See the code at github.com/vontell/MathIt-Android

Highlights

Highlights is an Android application that I worked on with my friends Cooper and Ali at YHack 2016. The application utilizes a machine learning backend which parses recently missed videos from your YouTube subscriptions to provide 10-30 second "summary" clips of that video. These small clips of relevant information are then displayed in a Snapchat-like way.

See the code at github.com/vontell/Highlights

FancySlider

While working on a fitness application, I needed a weight and height slider which had ticks, much like a ruler. I couldn't find any that fit my needs, so I made my own! My custom view is highly customizable (change the color, tick heights, width, precision, etc...), and is also quite efficient since it uses a RecyclerView to hold the slider components. The library can be easily imported as a Gradle / Maven dependency.

See the code at github.com/VP-Technologies/FancySlider
