Awards and Honors

As you can see below, I enjoy going to a lot of Hackathons; not only are they fun, but they are great ways to win some awesome prize!


Hackathon Awards

YHack 2016

2nd Place + AE Award

Highlights is an Android application that I worked on with my friends Cooper and Ali at YHack 2016. The application utilizes a machine learning backend which parses recently missed videos from your YouTube subscriptions to provide 10-30 second "summary" clips of that video. These small clips of relevant information are then displayed in a Snapchat-like way. For this we won second place, as well as a challenge for the best entertainment application.

See the code at github.com/vontell/Highlights

HackGT 2016

Best Use of AWS

Using TensorFlow, my friend Cooper and I built a website powered by TensorFlow to scrape data from Devpost, a popular hackathon hosting website. After parsing this data with a beefy AWS instance, we were able to predict and view the probability of winning a hackathon based on the number of team members, technologies used, and more. For this we won the Best Use of AWS award.

See the code at github.com/vontell/Judgd

SimHACK 2016

1st Place

My friend Michael and I created an event-crawling application for our dorm. The application was written for Android, and featured a Django backend. For this application, we won first place in our dorm's local hackathon.

See the code at github.com/vontell/SimCrawl

HackMIT 2017

Best Made from Scratch Award

My friend Cooper Pellaton, Ryan Petschek, and I created a platform for first responders to save victims of natural distasters. I was responsible for creating two Android applications, one of which was powered by AR. The backend was also powered by data analytics and some simple max likelihood functions.

See the entire project on Devpost

HackGT 2017

Top 8 and Best Smart City Hack

At HackGT I worked solo to create a smart city platform, which leveraged Twitter, GE's Current API, Heroku, Microsoft Azure Chat Bot Services, and Square Android APIs. The application allowed city citizens to collaborate with city officials to fix public issues.

See the project on Devpost
